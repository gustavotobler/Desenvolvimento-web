Lista de materiais escolares

Pequena explicação:

Prezados pais. Nossa escola formulou uma lista de materias
para nossos futuros alunos que ingressarão em nossa escola.
Está lista irá conter materias escolares para um aluno do fundamental 1.
Aqui irão alguns tópicos sobre o que será necessário:

  1. Materiais para a aula

    2. Os materias para aula, serão os materias básicos para um aluno realizar suas atividades.

    Como por exemplo fazer suas anotações.


    • Lápis
    • Canetas
    • Borracha
    • Apontadores
    • Régua
    • Tesoura
    • Cola
    • Caderno

  2. Materiais coletivos

    3. Os materias coletivos, serão os materias para aulas dinâmicas.

    Como por exemplo aulas de pintura.


    1. 5x Pacotes de Folha sulfite
    2. 3x Tinta guaxe
    3. 2x Pincel
    4. 3x Cartolina(Qualquer cor)
    5. 4x EVA
    6. 2x Folha A3

  3. Materiais de higiêne

    4. Os materias de higiêne, serão os que os alunos utilizaram para sua própria higiêne.

    Como por exemplo escovar os dentes.


    Estão inclusos:

    1. Escova de dente
    2. Pasta de dente
    3. Álcool 70%
    4. Toalha
    5. Papel toalha
    6. Sabonete líquido
